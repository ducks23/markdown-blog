# Neovim Commands

```
# open up mason
<leader> cm 

# rename variable
<leader> cr

# search for a string
<leader> sg

# see linting diagnostics
<leader> xx

# code actions to fix linting error
<leader> ca

# buffer right
]b

# buffer left
[b

# can control splits with ctrl + arrow keys
```
